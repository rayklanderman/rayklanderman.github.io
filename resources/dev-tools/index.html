Developer Tools & Cloud

Cloud platforms, hosting, CI/CD, version control, databases, and essential dev utilities.

  • 💻

    GitHub

    Unlimited public/private repos with CI/CD (Actions), static hosting (Pages), containers, and project management.

  • 💻

    GitLab

    Unlimited public/private repos with built-in CI/CD, container registry, and static hosting. Up to 5 collaborators.

  • ☁️

    Vercel

    Host static sites and serverless functions with a generous free tier. Native support for Next.js, React, and more.

  • 🌐

    Netlify

    Static hosting with form handling, serverless functions, split testing, and free SSL. 100GB bandwidth included.

  • ⛅

    Cloudflare

    Free CDN, DDoS protection, DNS, SSL, Workers (serverless), Pages (hosting), R2 storage, and D1 database.

  • 🌐

    Microsoft Azure Free Tier

    $200 credit for 30 days + always-free services: 1 B1S VM, 10 web apps, Functions, Cosmos DB 25GB, and more.

  • 🌐

    AWS Free Tier

    12 months free: t2.micro EC2, 5GB S3, 1M Lambda requests, 25GB DynamoDB, and 750 hrs RDS. Plus always-free offers.

  • 🌐

    Oracle Cloud Free Tier

    Always-free: 2 AMD VMs, 4 Arm cores/24GB RAM, 200GB storage, 10TB egress. Best free cloud compute offering.

  • 🗃

    Render

    Unified cloud for static sites, web services, and databases. Free tier includes static sites and PostgreSQL.

  • 🚆

    Railway

    Full-stack hosting platform. Free tier with $5 credit/month for deploying apps and databases.

  • 🗃

    Supabase

    Open-source Firebase alternative. PostgreSQL database with real-time subscriptions, auth, and storage. 2 free projects.

  • 🌲

    MongoDB Atlas

    Free shared cluster with 512MB storage. Great for NoSQL database needs in fullstack projects.

  • 🌐

    Cloudflare Pages

    Deploy static sites with 500 monthly builds, 100 custom domains, free SSL, and Workers integration.

  • 🚀

    VS Code

    Free, open-source code editor from Microsoft. Extensive extension ecosystem for every language and framework.
