Learning & Career Development

Free and accessible platforms for courses, certifications, coding practice, and career growth.

  • 📚

    Microsoft Learn

    Free interactive learning paths, modules, and certifications for Azure, Power Platform, AI, and more.

  • 📚

    freeCodeCamp

    Free full-stack coding curriculum with interactive challenges and certificates in web development, Python, and ML.

  • 📚

    Exercism

    Free coding practice with mentorship. 70+ programming languages with structured learning tracks.

  • 📌

    Roadmap.sh

    Community-driven developer roadmaps. Learn what to study to become a frontend, backend, DevOps, or AI engineer.

  • 🏫

    MIT OpenCourseWare

    Free course materials from MIT. Computer science, AI, data science, and engineering courses available.

  • 🌐

    Coursera

    Audit thousands of courses for free. Certificates available with financial aid. Partner universities worldwide.

  • 🌐

    edX

    Free courses from top universities including Harvard, MIT, and Stanford. Audit mode for most courses.

  • 🏫

    Harvard CS50

    Harvard's famous Introduction to Computer Science. Entirely free, with problem sets and a final project.

  • 📚

    The Odin Project

    Free, open-source full-stack curriculum for Ruby on Rails and JavaScript. Project-based learning approach.

  • 🌐

    Khan Academy

    Free educational platform with courses in math, science, computer programming, and economics.

  • 📚

    W3Schools

    Free web development tutorials and references. HTML, CSS, JavaScript, Python, SQL, and more with interactive editors.

  • 📚

    Refactoring Guru

    Free catalog of design patterns and refactoring techniques with code examples in multiple languages.
