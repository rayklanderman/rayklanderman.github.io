Data Science & Analytics

Databases, analytics platforms, visualization tools, and data APIs with free access tiers.

  • 📊

    Google BigQuery

    Serverless data warehouse. Free tier includes 1 TB of querying per month and 10 GB of storage.

  • 🗃

    Supabase

    Open-source PostgreSQL database with real-time subscriptions, authentication, and storage. 2 free projects.

  • 🌲

    MongoDB Atlas

    Free shared cluster (512MB-5GB) for NoSQL document storage. Ideal for analytics and application data.

  • 🌐

    PlanetScale

    MySQL-compatible serverless database platform with branching workflows. Free tier includes 5GB storage.

  • 📈

    Cube

    Semantic layer for data analytics. Cube Cloud free tier includes 1,000 queries per day.

  • 📚

    Deepnote

    Collaborative data science notebook environment with real-time collaboration and cloud execution.

  • 💻

    Hex

    Collaborative data platform for notebooks, data apps, and knowledge libraries. Free community tier (5 projects).

  • 📈

    Redash

    Open-source dashboarding and visualization tool. Connect to any data source and build interactive dashboards.

  • 📈

    Evidence

    Open-source SQL-based analytics tool. Build reports and dashboards with markdown and SQL.

  • 🏆

    Kaggle

    Data science community with free datasets, competitions, notebooks (4 CPU/30GB RAM), and learning courses.

  • 📊

    Power BI

    Microsoft's business analytics tool. Free desktop app for data visualization and report creation.

  • 📊

    Tableau Public

    Free data visualization tool. Create and share interactive dashboards publicly on the web.
